## Day-03
### Understand JSX
It's not HTML - it's JavaScript with a readable syntax
Use className, not class
Only return one parent element

### Learn the rules
Self-close all tags
Wrap JavaScript inside ```{}```
Use ternary operators or if statements to conditionally render
